# Privacy Policy for DSA Reviser

DSA Reviser does not collect, store, or share any personal user data.

## Data Usage

This extension only accesses and stores the following information locally on the user's device:
- LeetCode problem title and URL
- User-defined difficulty levels
- Spaced repetition schedules

All data is stored using Chrome's local storage API and is never transmitted to external servers.

## Permissions

- Storage: Used to store saved problems and schedules locally
- Active Tab & Scripting: Used to access and interact with LeetCode problem pages
- Alarms & Notifications: Used to send revision reminders

## Third-Party Services

This extension does not use any third-party services, analytics, or external APIs.

If you have any questions, please open an issue on the GitHub repository.
