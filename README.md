# DSA Reviser – Spaced Repetition for LeetCode

A Chrome extension that helps you retain DSA problems using spaced repetition.

## Features

- Save problems directly from LeetCode
- Custom difficulty tagging
- Smart spaced repetition (2, 7, 10 days or custom)
- Revision reminders every time you open the browser
- Clean popup dashboard
- Confetti on successful submissions 🎉

## Why this?

Most people solve problems and forget them.  
This extension ensures you **revise at the right time** so concepts stick.

## Privacy

All data is stored locally using Chrome storage.  
No data is collected or sent externally.

## Tech Stack

- HTML, CSS, JavaScript
- Chrome Extension APIs (storage, alarms, notifications)

## Installation

1. Go to `chrome://extensions/`
2. Enable Developer Mode
3. Click "Load unpacked"
4. Select this folder
